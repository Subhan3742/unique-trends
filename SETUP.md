# UniqueTrends.pk Setup Instructions

## Logo Setup

Please add your logo image to the project:

1. Save your logo image as `logo.png` in the `my-app/public/` folder
2. The logo should be in PNG format with transparent background for best results
3. Recommended size: 800x400 pixels or similar aspect ratio

## Running the Development Server

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
